# TypeScript

## Check the version
- ```tsc -v```

## Configuration File
- ```tsc --init``` -> configure the compiler
- ```tsc``` -> compile the code (dist folder)


# Built-in Types
- ```any```, ```unknown```, ```never```, ```enum``` & ```tuple```


## Shortcuts
- E.g. ```tsc & node/dist index.js``` -> Compile & Run
